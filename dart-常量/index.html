# 常量

本文作者：阳九五 (opens new window)

本站地址：https://blog.56321654.xyz (opens new window)

常量有两种 final 和 const

# 相同点

# 声明类型可以省略

final String a = 'Young';
final a = 'Young';

const String a = 'Young';
const a = 'Young';

# 不能使用var去申明

var会推导变量的类型,不是数据类型

final var a = 'Young'; // 报错

const var a = 'Young'; // 报错

# 不同点

# 赋值的时候不同

final 运行时赋值、可以先声明后赋值,只能被赋值一次
const 编译时赋值、声明的同时就要赋值,只能被赋值一次

final dt = DateTime.now(); //正常

final dt  // 正常
dt = DateTime.now(); //正常 

const dt  // 报错
const dt = DateTime.now(); // 报错
const dt = "2024-10-10"; // 正常

# 不可变性可传递

final 值可以修改
const 值不能修改

final List ls = [11, 22, 33];
ls[1] = 44;  // 正常

const List ls = [11, 22, 33];
ls[1] = 44;  // 报错

# 内存中重复创建

final 不存在内存复用 const 存在内存复用

final a1 = [11 , 22];
final a2 = [11 , 22];
print(identical(a1, a2));  // 显示flase

const a1 = [11 , 22];
const a2 = [11 , 22];
print(identical(a1, a2));  // 显示true
