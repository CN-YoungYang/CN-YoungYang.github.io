# ~ 位非运算

二进制数逐位进行逻辑非运算

0 1 0 0 1       最高位代表正负 0非负数,1负数
0 0 1 1 0       补码
1 1 0 0 1       取反
1 1 0 1 0       加1
--------
1 1 0 1 0       -10

print(~9); // 输出 -10

# << 左移运算符

进制向左移N位数

0 0 0 1          1 二进制
0 0 1 0          左移一位 2
0 1 0 0          左移二位 4
1 0 0 0          左移三位 8

print( 1 << 1 );  // 输出2
print( 1 << 2 );  // 输出4
print( 1 << 3 );  // 输出8

进制向左移N位数

0 0 0 1          右移三位 1
