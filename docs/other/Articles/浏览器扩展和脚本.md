## 浏览器扩展

### Dark Reader

**下载地址**: [https://darkreader.org/](https://darkreader.org/)

**功能描述**: 适用于所有网站的暗色主题，保护眼睛，减少屏幕蓝光。

**使用建议**: 可以自定义亮度、对比度和灰度，支持自动切换日间/夜间模式。

---

### Global Speed

**下载地址**: [Chrome Web Store](https://chromewebstore.google.com/detail/global-speed-%E8%A7%86%E9%A2%91%E9%80%9F%E5%BA%A6%E6%8E%A7%E5%88%B6/jpbjcnkcffbooppibceonlgknpkniiff)

**功能描述**: 通用视频速度控制，支持所有视频网站。

**使用建议**: 可以在 0.1x 到 16x 之间调整播放速度，支持快捷键控制。

---

### History Plus

**下载地址**: [Chrome Web Store](https://chromewebstore.google.com/detail/history-plus/kloodnjmhgicecceindgbfpjencnhajh)

**功能描述**: 使 Chrome 历史记录超过 3 个月，提供全面的筛选功能。

**使用建议**: 支持按时间、域名、关键词等多维度筛选历史记录。

---

### OneTab

**下载地址**: [Chrome Web Store](https://chromewebstore.google.com/detail/onetab/chphlpgkkbolifaimnlloiipkdnihall)

**功能描述**: 标签页快速合并，一键将所有标签页保存为一个列表。

**使用建议**: 适合标签页过多时使用，可以节省内存，方便恢复。

---

### Proxy SwitchyOmega

**下载地址**: [Chrome Web Store](https://chromewebstore.google.com/detail/proxy-switchyomega-3-zero/pfnededegaaopdmhkdmcofjmoldfiped)

**功能描述**: 切换多个代理设置，支持 PAC、SOCKS5、HTTP 等多种代理协议。

**使用建议**: 适合需要频繁切换代理的用户，支持配置文件导入导出。

---

### SingleFile

**下载地址**: [Chrome Web Store](https://chromewebstore.google.com/detail/singlefile/mpiodijhokgodhhofbcjdecpffjipkle)

**功能描述**: 保存网页为单个 HTML 文件，包含所有资源（图片、CSS、JS）。

**使用建议**: 适合保存需要离线查看的网页，支持批量保存。

---

### uBlock Origin

**下载地址**: [Chrome Web Store](https://chromewebstore.google.com/detail/ublock-origin/cjpalhdlnbpafiamejdnhcphjbkeiagm)

**功能描述**: 强大的广告拦截器，同时可以阻止跟踪器和恶意软件。

**使用建议**: 
- 新版 Chrome 使用方法：[YouTube 教程](https://www.youtube.com/watch?v=AXN-OcOeeHM)
- 可以自定义过滤规则，支持元素选择器模式

---

### 欧路翻译

## 使用建议

1. **扩展选择**: 扩展装多了会影响浏览器性能，建议只安装必要的扩展，其他功能尽量使用脚本实现。
2. **脚本管理**: 定期检查脚本更新，确保功能正常。
3. **安全注意**: 只从可信来源下载扩展和脚本，注意保护隐私。
4. **性能优化**: 可以使用扩展管理器来禁用不常用的扩展，提升浏览器性能。
