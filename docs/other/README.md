## 🚀 快捷工具

### Windows 激活脚本

使用 MAS (Microsoft Activation Scripts) 远程脚本激活 Windows 系统：

```powershell
# 1. 首先设置执行策略
Set-ExecutionPolicy RemoteSigned

# 2. 执行激活脚本
irm https://get.activated.win | iex
```

**使用说明**：
- 以管理员身份运行 PowerShell
- 执行上述命令即可开始激活流程
- 脚本会自动检测并激活你的 Windows 系统
