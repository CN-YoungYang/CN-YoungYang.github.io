# 使用建议

  1. 综合搜索：优先使用安娜的档案、鸠摩搜书等聚合类网站进行搜索
  2. 技术书籍：搬书匠、码农之家对国内技术书籍收录较全
  3. 古籍文献：书格、国学大师网适合查找古典文献
  4. 学术资源：Library Genesis、科学文库适合学术研究

注意：部分网站可能需要科学上网访问，请根据实际情况选择合适的资源站。
