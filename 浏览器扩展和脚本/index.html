# 浏览器扩展和脚本

本文作者：阳九五 (opens new window)

本站地址：https://blog.56321654.xyz (opens new window)

本文整理了常用的浏览器扩展和油猴脚本，帮助你提升浏览器使用效率。扩展装多了会卡，建议尽量使用脚本。

# 浏览器扩展

# Dark Reader

下载地址: https://darkreader.org/ (opens new window)

功能描述: 适用于所有网站的暗色主题，保护眼睛，减少屏幕蓝光。

使用建议: 可以自定义亮度、对比度和灰度，支持自动切换日间/夜间模式。

# Global Speed

下载地址: Chrome Web Store (opens new window)

功能描述: 通用视频速度控制，支持所有视频网站。

使用建议: 可以在 0.1x 到 16x 之间调整播放速度，支持快捷键控制。

# History Plus

下载地址: Chrome Web Store (opens new window)

功能描述: 使 Chrome 历史记录超过 3 个月，提供全面的筛选功能。

使用建议: 支持按时间、域名、关键词等多维度筛选历史记录。

# OneTab

下载地址: Chrome Web Store (opens new window)

功能描述: 标签页快速合并，一键将所有标签页保存为一个列表。

使用建议: 适合标签页过多时使用，可以节省内存，方便恢复。

# Proxy SwitchyOmega

下载地址: Chrome Web Store (opens new window)

功能描述: 切换多个代理设置，支持 PAC、SOCKS5、HTTP 等多种代理协议。

使用建议: 适合需要频繁切换代理的用户，支持配置文件导入导出。

# SingleFile

下载地址: Chrome Web Store (opens new window)

功能描述: 保存网页为单个 HTML 文件，包含所有资源（图片、CSS、JS）。

使用建议: 适合保存需要离线查看的网页，支持批量保存。

# uBlock Origin

下载地址: Chrome Web Store (opens new window)

功能描述: 强大的广告拦截器，同时可以阻止跟踪器和恶意软件。

使用建议:

  • 新版 Chrome 使用方法：YouTube 教程 (opens new window)
  • 可以自定义过滤规则，支持元素选择器模式

# 欧路翻译

下载地址: Chrome Web Store (opens new window)

功能描述: 网页翻译插件，支持双语对照显示。

使用建议:

  • 建议使用 oaipro 的 gpt-5-mini 翻译接口
  • 或从火山方舟 (opens new window)获取参数自己填

# Bitwarden 密码管理器

下载地址: Chrome Web Store (opens new window)

功能描述: 开源免费的密码管理器，支持跨平台同步，提供密码生成、自动填充等功能。

使用建议:

  • 支持多设备同步，数据加密存储
  • 可以生成强密码，避免重复使用密码
  • 支持导入其他密码管理器的数据
  • 建议开启两步验证增强安全性

# AutoVerify

下载地址: Chrome Web Store (opens new window)

功能描述: 邮箱自动验证工具，可以自动跳过邮箱验证码步骤，快速完成邮箱验证。

使用建议:

  • 支持多个邮箱服务提供商
  • 可以自动识别并填写验证码
  • 适合需要频繁注册账号的用户
  • 注意保护个人隐私信息

# 篡改猴测试版

下载地址: Chrome Web Store (opens new window)

功能描述: 使用油猴脚本，一些脚本必须用测试版才能安装。

使用建议: 油猴脚本的必备扩展，支持自定义脚本管理。

# 禁用 WebRTC

下载地址: 禁用 WebRTC (opens new window)

功能描述: 关闭 WebRTC，阻止直播平台偷流量。

使用建议: 适合使用流量有限制的网络环境，可以防止意外流量消耗。

# 使用建议

  1. 扩展选择: 扩展装多了会影响浏览器性能，建议只安装必要的扩展，其他功能尽量使用脚本实现。
  2. 脚本管理: 定期检查脚本更新，确保功能正常。
  3. 安全注意: 只从可信来源下载扩展和脚本，注意保护隐私。
  4. 性能优化: 可以使用扩展管理器来禁用不常用的扩展，提升浏览器性能。
